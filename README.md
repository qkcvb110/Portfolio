>

## **👨🏻‍💻 기능 구현**

### **1. CAN,CAN-FD Communication**

- 프로젝트에 사용된 주 통신으로 CAN(Controller Area Network)통신을 사용
- STM 간은 CAN-FD, Raspberry Pi와 STM 간의 통신은 CAN을 사용하여 통신

### **2. RF Communication**

- 전동자동차를 원격으로 조종하기 위한 통신으로 RF 통신을 사용
